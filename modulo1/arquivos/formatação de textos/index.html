Formatação de Textos

Negrito / Destaque

Para colocar uma palavra em negrito desta forma utiliza-se a tag <strong>

Itálico / Ênfase

Para colocar uma palavra em italico ou em ênfase num texto desta forma utiliza-se a tag <em>

Texto Marcado

Podemos criar também um texto marcado desta forma com a tag <mark>

Texto Deletado

Para "deletar" um texto desta forma, utiliza-se a tag <del>

Texto Sublinhado / Inserido

Para colocar um texto sublinhado desta forma utiliza-se a tag <ins>

Texto Sobrescrito

Para colocar um termo sobrescrito como neste caso: x4+20 = 8, utiliza-se a tag <sup>

Texto Subescrito

Para colocar um termo subescrito como neste caso: H20, utiliza-se a tag <sub>

Código Fonte / Pré-Formatação
