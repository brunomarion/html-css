O que são formulários?

É um local onde o usuário pode colocar dados.

Segue abaixo o exemplo mais básico de Formulários:


EXISTEM DOIS MÉTODOS PARA ENVIO DE FORMULÁRIOS: GET E POST

GET: método padrão de envio de dados, aparecendo na URL. Isso deixa explícito na URL do site os dados enviados.

POST: método de envio de dados, não aparecendo na URL. Isso deixa inplícito os dados enviados, mas não protege os dados por causa disso, dá para encontrar indo nos "headers", do Network no inspecionar do arquivo destino "action".


MAS QUANDO USAR CADA UMA?

GET:

  • Normamente usa-se para formulários com envio de dados em campos mais simples, sem ter algo tão importante, como nome, sobrenome, data de nascimento, etc...
  • Também usa-se bastante para compartilhamento de URL, como o google pro exemplo, pois os dados de pesquisa ficam na URL.
  • SÓ PODE ENVIAR ATÉ 3000 BITS (LIMITE), o que dá aproximadamento 3000 letras em dados enviados.

POST:

  • Quando se trabalha com campos de dados mais sensíveis para o usuário, como senhas, número de cartão de crédito, cpf...
  • Protege "um pouco" mais o envio desses dados, pois fica implícito e não aparece na URL, por exemplo
  • Se os dados ultrapassarem o limite de 3000 BITS (LIMITE)
