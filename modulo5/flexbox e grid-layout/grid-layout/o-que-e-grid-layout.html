O que é um Grid Layout?

É um sistema de criação de Layouts (nterfaces de Usuário), baseado em uma grade bidimensional

Propriedades de um GridLayout:

  • Grid Conteiner: É o conteiner Pai, ele quem engloba todos os grid Items.
  • Grid Item: São todas as junções do Grid, cada encontro entre linhas e colunas.
  • Grid Tracks: Podem ser divididas entre RowTrack e ColumnTrack.
  • Grid Area: Como o nome já diz são um conjunto de Grid Items.
  • Grid-Lines: São as linhas que compõe o Grid-Container, sendo positivas (para direita e para baixo) e negativas (para esquerda e para cima).

Propriedades de um Grid Container:

  • Display: Grid; (Transforma este elemento em um container do tipo Grid, desbloqueando todas as propriedades de um grid container).
  • Grid-Tamplate-Columns: auto auto auto (define o tamanho das colunas do grid. AUTO siginifica que vai preencher todo o espaço igualmente). Pode-se usar a função Repeat().
  • Grid-Tamplate-Rows: 100px 100px 100px (define o tamanho das linhas do grid. AUTO siginifica que vai preencher todo o espaço igualmente). Pode-se usar a função Repeat().
  • Grid-Gap: 20px (espaçamento entre os gridItems).
  • Align-Items:Alinha os items verticalmente individualmente, dentro de suas grades. Podem ser: Stretch (Padrão) / Start (Alinha no Começo do Item) / Center (Alinha no centro do Item) / End (Alinha no Final do Item)
  • Justify-Items:Alinha os items individualmente horizontalmente, dentro de suas grades. Podem ser: Stretch (Padrão) / Start (Alinha no Começo do Item) / Center (Alinha no centro do Item) / End (Alinha no Final do Item)
  • Align-Content:Alinha a posição do conteúdo inteiro (conjunto de items) verticalmente, dentro do container. Podem ser: Stretch (Padrão) / Start (Alinha no Começo do Container) / Center (Alinha no centro do Container) / End (Alinha no Final do Container) /Space-Between / Space-Around / Space-Evenly.
  • Justify-Content:Alinha a posição do conteúdo inteiro (conjunto de items) horizontalmente, dentro do container. Podem ser: Stretch (Padrão) / Start (Alinha no Começo do Container) / Center (Alinha no centro do Container) / End (Alinha no Final do Container) /Space-Between / Space-Around / Space-Evenly.
  • Grid-Auto-Row / Grid-Auto-Column: Alinhar um novo conteudo adicionado ao grid com o tamanho definido.

Short-Hands propriedades Grid-Conteiner:

  • Grid-Tamplate: Grid-Tamplate-Rows / Grid-Tamplate-Columns. Com função Repeat(quantidade de repetição, tamanho): Grid-Tamplate: repeat(3, 100px) / repeat(3, auto)
  • Place-Items: Align-Items Justify-Items
  • Place-Content: Align-Content Justify-Content
