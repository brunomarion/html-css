Flex-Shrink e Flex-Grow

Propriedade que define se o elemento vai encolher ou crescer, conforme o tamanho do conteiner

ATENÇÃO:
0 -> define que não vai crescer

1-> Define que vai crescer

Qualquer numero acima de 1, vai crescer (grow) ou vai diminuir (shrink) na proporção dos outros items

Existe o ShortHand: (Flex: flex-grow + flex-shrink + flex-basis)

Outros valores para flex bastante utilizados são:

  • Flex: Initial (Flex: 0 1 auto - valores padrões de cada propriedade)

  • Flex: None (Flex: 0 0 auto) - para o item nao flexível

  • Flex: auto (Flex: 1 1 auto) - para um item totalmente flexível

  • Flex: 1 / Flex: 2 / Flex: 3 ... (Flex: 1/2/3 1 auto) - muda somente o Grow, o restante fica padrão
