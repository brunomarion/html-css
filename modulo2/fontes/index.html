Categorias de Fontes

Tipos de fontes:

  • Serifadas
  • Sans-Serif (não serifadas)
  • Monoespaçada
  • Display
  • HandWriting

Tipos de medidas de fontes:

  • Medidas Absolutas:
    • cm, mm, in, px, pt, pc

  • Medidas Relativas:
    • em, ex, rem, vw, vh, %

As medidas mais usadas são px(pixels) e em(relativo), onde 16px é o tamanho de fonte padrão do navegador

16px = 1em (geralmente)

Para usar fontes incorporadas da internet recomenda-se o: google.fonts.com

Para usar fontes baixadas da internet existem vários sites mas um deles é o: dafont.com

Para extrair fontes usadas em imagens, sites indicados são: https://www.whatfontis.com/, https://www.fontsquirrel.com/, https://www.myfonts.com/
