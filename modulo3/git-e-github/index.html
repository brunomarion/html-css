GIT E GITHUB DIFERENÇAS

GIT

Git é uma ferramenta de versionamento de código que instalamos na nossa máquina. Para maior controle durante a evolução e execução no desenvolvimento de um software/sistema/site, por meio de um repositório local


GITHUB

GitHub é uma ferramenta/ambiente usado para controlar o nosso código num repositório remoto

Push -> É usado para atualizar nosso repositório remoto, atualizando ele com as mudanças feitas em nosso repositório local

Pull -> É usado para atualizar nosso repositório local, atualizando ele com as mudanças feitas em nosso repositório remoto
